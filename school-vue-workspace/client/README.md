# School Vue Project

This is a Vue.js application for managing student score entries. The project is structured to facilitate easy development and maintenance.

## Project Structure

```
school-vue-workspace
├── school-vue-workspace.code-workspace
├── client
│   ├── package.json
│   ├── vite.config.js
│   ├── index.html
│   ├── src
│   │   ├── main.js
│   │   ├── App.vue
│   │   └── components
│   │       └── ScoreEntry.vue
│   ├── public
│   │   └── favicon.ico
│   └── README.md
├── .vscode
│   └── settings.json
├── package.json
└── README.md
```

## Getting Started

To get started with the project, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd school-vue-workspace
   ```

2. **Navigate to the client directory:**
   ```bash
   cd client
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Run the application:**
   ```bash
   npm run dev
   ```

## Features

- User-friendly interface for entering and managing student scores.
- Responsive design for accessibility on various devices.
- Print functionality for score sheets.
