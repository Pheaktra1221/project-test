# School Vue Project

This project is a Vue.js application designed for managing student score entries. It provides a user-friendly interface for teachers to input and manage scores for different classes and subjects.

## Project Structure

│   ├── src

│   │   ├── App.vue
│   │   └── components
│   │       └── ScoreEntry.vue

## Getting Started

To get started with this project, follow these steps:

1. **Clone the repository**:
   ```
   git clone <repository-url>
   cd school-vue-workspace
   ```

2. **Install dependencies**:
   Navigate to the `client` directory and run:
   ```
   npm install
   ```

3. **Run the application**:
   Start the development server by running:
   ```
   npm run dev
   ```

4. **Open the application**:
   Open your browser and go to `http://localhost:3000` (or the port specified in your terminal).

## Features

- User-friendly interface for entering student scores.
- Dynamic filtering and searching of students and subjects.
- Print preview functionality for score sheets.
- Responsive design for various devices.
